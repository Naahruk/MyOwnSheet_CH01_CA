HTML semantics

Basic semantic tags

Tag Name Function
<header> Header First elements to introduce the page, i. e., navegation options, logo, etc.
<nav> Navigation Links to content in the same page, documents or external sources.
<main> Main Area to put the main content of the page, the central information.
<footer> Footer Information referred to credits, copyright, author's name.
<section> Section Elements with the same content, like chapters, headings, etc.
<article> Article Parts of the text like blogs, comments, etc.
<aside> Aside Additional information related to the main topic.
<figure> Figure Contents all kinds of media.
<figcaption> Figure Caption Adds a description of the media in the page.
<video> Video Appears a media player reproducer for control de video.
<audio> Audio Appears controls to use in the audio file.
<embed> Embed Adds different content (media, audio, etc.) to the page
